# node-express-angular
本项目是由nodeJS+express做的后端，模板用的jade，前端用的angularJS+bootstrap  ，mongodb做的数据库。

主要实现 增 删 查 改功能.  
socket 实现聊天室.
引入百度Echarts做了个图表例子。
本项目纯属个人娱乐。如果有什么问题可以联系我QQ:36637737


## 安装
首先保证你安装好nodeJS

下载后 需要安装依赖

先npm安装插件
npm install

全局安装bower
npm install bower -g 

使用bower安装插件
bower install

## 启动
node app

数据库未上传-mongodb
